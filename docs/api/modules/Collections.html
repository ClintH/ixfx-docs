Const add

  • add<K, V>(map: ReadonlyMap<K, V>, ...data: EitherKey<K, V>): ReadonlyMap<K, V>

  • Adds data to a map, returning the new map.

    Can add items in the form of [key,value] or {key, value}.

    example

    These all produce the same result

    map.set(`hello`, `samantha`);
    map.add([`hello`, `samantha`]);
    map.add({key: `hello`, value: `samantha`})

    Type parameters

    • K

    • V

    Parameters

    • map: ReadonlyMap<K, V>

      Initial data

    • Rest ...data: EitherKey<K, V>

      One or more data to add in the form of [key,value] or {key, value}

    Returns ReadonlyMap<K, V>

    New map with data added

Const groupBy

  • groupBy<K, V>(array: readonly V[], grouper: (item: V) => K): Map<K, V[]>

  • Groups data by a grouper function, returning data as a map with string keys and array values.

    example
    const data = [
     { age: 39, city: `London` }
     { age: 14, city: `Copenhagen` }
     { age: 23, city: `Stockholm` }
     { age: 56, city: `London` }
    ];
    const map = groupBy(data, item => data.city);

    Returns a map:

    London: [{ age: 39, city: `London` }, { age: 56, city: `London` }]
    Stockhom: [{ age: 23, city: `Stockholm` }]
    Copenhagen: [{ age: 14, city: `Copenhagen` }]

    Type parameters

    • V

    Parameters

      Data to group

      Function that returns a key for a given item

        • (item: V): K

    Returns Map<K, V[]>

Const hasAnyValue

  • hasAnyValue<K, V>(map: ReadonlyMap<K, V>, value: V, comparer: IsEqual<V>): boolean

  • Returns true if any key contains value, based on the provided comparer function. Use hasKeyValue if you only want to find a value under a certain key.

    Having a comparer function is useful to check by value rather than object reference.

    example

    Finds value samantha, using string equality to match

    hasAnyValue(map, `samantha`, (a, b) => a === b);

    Type parameters

    • V

      Map to search

    • value: V

      Value to find

    • comparer: IsEqual<V>

      Function that determines matching

    Returns boolean

    True if value is found

Const queue

  • queue<V>(opts?: QueueOpts, ...startingItems: readonly V[]): Queue<V>

  • Returns an immutable queue

    let q = queue();           // Create
    q = q.enqueue(`a`, `b`);   // Add two strings
    const front = q.peek();    // `a` is at the front of queue (oldest)
    q = q.dequeue();           // q now just consists of `b`

    Type parameters
